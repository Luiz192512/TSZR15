## 🔍 COMO VERIFICAR SE O DOMÍNIO ESTÁ DISPONÍVEL

### Método 1: Direto no Namecheap
1. Acesse https://www.namecheap.com
2. Cole o domínio na barra "Find your domain"
3. Clique em "Search"
4. Se mostrar preço → **DISPONÍVEL** ✅
5. Se mostrar "Taken" → **INDISPONÍVEL** ❌

### Método 2: Verificar WHOIS
1. Acesse https://www.whois.com
2. Cole o domínio
3. Se houver dono registrado → **INDISPONÍVEL** ❌
4. Se disser "No Found" → **DISPONÍVEL** ✅

### Método 3: Bulk Check (verificar múltiplos)
1. https://bulkwhois.com
2. Cole vários domínios (um por linha)
3. Vê todos os status de uma vez

## 🔗 CONECTAR DOMÍNIO À SUA HOSPEDAGEM

### Se usar **Vercel:**
```
1. Vá para Namecheap > Manage > DNS
2. Procure "nameserver"
3. Altere para:
   - ns1.vercel.com
   - ns2.vercel.com
4. Salve (pode levar 24-48h)
```

### Se usar **Cloudflare:**
```
1. Crie conta Cloudflare
2. Adicione site
3. Copie nameservers Cloudflare
4. Vá para Namecheap > DNS
5. Cole nameservers
6. Salve (pode levar 24-48h)
```

### Se usar **DigitalOcean:**
```
1. DigitalOcean > Networking > Domains
2. Copie nameservers
3. Namecheap > DNS > mude para nameservers DO
4. Salve
```
